Module 12: AWS Snow Family

12.1 What is the AWS Snow Family?

The AWS Snow Family is a collection of physical devices that help you transfer large amounts of data into and out of AWS, and perform compute operations in rugged, disconnected, or mobile environments. These services are designed for situations where network connectivity is limited, unreliable, or non-existent, making traditional internet-based data transfer impractical.

Key Use Cases:

  • Large-scale data migration: Moving petabytes or exabytes of data to AWS.
  • Edge computing: Running applications in remote or harsh environments.
  • Disconnected operations: Performing compute and storage tasks without continuous internet access.

12.2 AWS Snowball

AWS Snowball is a petabyte-scale data transport solution that uses secure appliances to transfer large amounts of data into and out of AWS. It comes in two main options:

  • Snowball Edge Storage Optimized: Provides up to 80 TB of usable storage capacity for block volumes and Amazon S3 compatible object storage, and 40 vCPUs. Ideal for large-scale data migrations and recurring transfer workflows, as well as local storage and compute needs.
  • Snowball Edge Compute Optimized: Provides 42 TB of usable storage capacity for block volumes and Amazon S3 compatible object storage, and 52 vCPUs with an optional GPU. Ideal for advanced use cases like machine learning, full-motion video analysis, analytics, and local compute.

How it Works:

  1. You request a Snowball device from the AWS console.
  2. AWS ships the device to you.
  3. You connect the device to your local network and transfer data using a client or S3-compatible interface.
  4. You ship the device back to AWS.
  5. AWS uploads the data to your S3 bucket.

12.3 AWS Snowcone

AWS Snowcone is the smallest member of the AWS Snow Family, designed for edge computing, data migration, and content distribution. It's a portable, rugged, and secure device that can be used in harsh environments.

Key Features:

  • Small and Portable: Weighs about 4.5 pounds (2.1 kg).
  • Storage: 8 TB of usable storage.
  • Compute: 2 vCPUs and 4 GB of memory.
  • Connectivity: Can connect via Wi-Fi or wired network.
  • Use Cases: Data collection from remote sites, IoT data aggregation, content distribution to mobile environments.

Snowcone can be shipped back to AWS, or you can use AWS DataSync to transfer data online if network connectivity becomes available.

12.4 AWS Snowmobile

AWS Snowmobile is an exabyte-scale data transfer service used to move extremely large amounts of data (up to 100 petabytes per Snowmobile) to AWS. It comes in a 45-foot ruggedized shipping container, pulled by a semi-trailer truck.

Key Features:

  • Exabyte Scale: Designed for transferring massive datasets (e.g., entire data centers).
  • High Capacity: Up to 100 PB per Snowmobile.
  • Secure: Physically secure, temperature-controlled, water-resistant, and includes GPS tracking and video surveillance.
  • Use Cases: Migrating entire data centers, shutting down large on-premises storage arrays.

Snowmobile is used for one-time, massive data transfers where network bandwidth is insufficient or too costly.
